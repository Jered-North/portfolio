Rowan's Art

Web App/Ecommerce site for Rowan Barson's art




Client: DevMountain Solo Project

My Role: Creator, Developer

View Git Repository View Online


A mockup of what a gallery site for Rowan Barnson would look like where she is able to display her pieces of art and sell art pieces that she has for sale.

The site uses sessions with the cart system so that there is no need for logins for seperate users if someone wants to buy art.
